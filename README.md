## JavaFX

Plusieurs problèmes peuvent survenir lors de l'installation de JavaFX sur macOS avec puces M1/M2/M3/M4.

**Version du SDK utilisée :** 17.0.17 x64 de Gluon  
Lien : [https://gluonhq.com/products/javafx/](https://gluonhq.com/products/javafx/)  
> Cette version comporte quelques bugs et peut être lente pour initialiser un projet Java.

### Solution simple

1. Télécharger un JDK incluant JavaFX, par exemple un build Azul :  
   [https://gluonhq.com/products/javafx/](https://gluonhq.com/products/javafx/)
2. Configurer le nouveau JDK dans les paramètres d'Eclipse.
3. Associer ce JDK au projet Java.

!!  [cette solution solution a été trouver sur Stack Overflow ](https://stackoverflow.com/questions/71622429/is-it-possible-to-run-javafx-with-java-11-on-m1-mac)
